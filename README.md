# zyBook-Autocomplete
This script is a browser-based automation tool designed to assist with answering interactive zyBooks questions. It scans visible questions on the page, detects available answer choices, and automatically attempts them one by one until the correct answer is identified.

# Installation

1. Install Tampermonkey
Download and install the Tampermonkey browser extension:
https://www.tampermonkey.net

2. Install the script
Click [here](https://github.com/NotMathew/zyBook-Autocomplete/raw/refs/heads/main/zyBook-Autocomplete.user.js) to install the script.

3. Open zyBooks
Go to https://www.zybooks.com

The script will run automatically on supported pages.
